## Prerequisites

- Ruby (version 2.6.3)
- Rails (version 6.0.2)
- SQLite3 

## Getting Started

### Installation

1. Clone the repository:

    ```bash
    git clone git@github.com:mskilab-org/case-report-api.git
    ```

2. Install dependencies:

    ```bash
    bundle install
    ```

3. Set up the database:

    ```bash
    rails db:migrate
    ```

### Usage

Start the Rails server:

```bash
rails server
